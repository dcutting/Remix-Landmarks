!1 LandmarkList

Fetch and display a list of landmarks.

First set up some data for the landmark service.

|Fixture Landmark Service |
|name  |latitude|longitude|
|London|51.5    |0.1      |
|Paris |48.8    |2.3      |

Then start the Landmark List Coordinator.

|Ordered query: landmark list rows|
|row text                         |
|LONDON (51.5, 0.1)               |
|PARIS (48.8, 2.3)                |
